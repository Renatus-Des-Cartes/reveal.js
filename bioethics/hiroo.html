0. なぜ倫理学を学ぶのか

ノーラの事例

［引用ブロック・後で記入］

ヘンリク・イプセン『人形の家』より

ノーラの選択と対立

  • 夫（ヘルメル・トルヴァル）の「人形」として生きてきたノーラ
  • ノーラは、「こうしなくちゃならない」と確信し、夫も子どもも捨てて家を出る
  • ヘルメルは、信仰・道徳・法律・社会などを根拠に、ノーラを止めようとする
  • ノーラは、「自分自身に対する義務」を理由に、これらの根拠を拒絶する

「外側の規範」か、「内側の確信」か

  • 信仰・道徳・法律・社会の尺度ではすべきでない（悪）とされ得る行為
  • しかしノーラは、それをすべき（善・正しい）と確信

ノーラの行為を支えるのは、外側の規範ではなく、内側の確信＝自分自身への配慮。

→ ここに「倫理の尺度」が立ち上がる。

倫理＝いくつもの「尺度」のひとつ

  • 宗教的尺度・道徳的尺度・法的尺度・社会的尺度…
  • 倫理学のモチベーション：
    • 他の尺度ではBadでも、実はGoodではないか → 肯定（例：ノーラ、『高瀬舟』的ケース、ダークヒーロー）
    • 他の尺度ではGoodでも、実はBadではないか → 批判（例：ハラスメント、ELSI/RRI、環境破壊）

医療における「尺度」の衝突

  • 倫理学は、広く受け入れられた「尺度」の抑圧や不備を点検する
  • 医療では、医学的善悪の尺度と患者個人の幸不幸の尺度がしばしば衝突
  • 倫理学的観点は、患者に寄りそう対応へのヒントになり、倫理的リスクの回避にも資する

I. 導入

本章の目標

  • 倫理 / 倫理学の基本概念を確認
  • 理論 義務論 / 帰結主義の要点を理解
  • 実践 他者理解と対話の枠組み＝ナラティブ倫理を掴む

倫理・規範・道徳

  • 倫理＝ものごとの「よしあし」
  • 社会に共有された考え＝規範（社会規範）
  • 合意の強弱：例）「殺人は悪」⇄「安楽死」
  • 倫理学＝規範はなぜ正しいのか／なぜ従うべきかを探究
  • 道徳（モラル）＝「慣習」が起源。法的罰は無いが社会的非難の対象になり得る

倫理学の二つの道筋

規範倫理学

  • 規範そのものの妥当性を論証
  • 例：「安楽死は◯◯の根拠で正しい」

記述倫理学

  • 規範に関する事実を実証
  • 例：「◯◯％が安楽死に賛成」

看護倫理では両輪が不可欠。

倫理的判断の基準：外からの強制力

  • 法律：社会秩序のための強制力（罰則）
  • 倫理綱領／ガイドライン：専門職団体の規則（ソフトロー）

内的動機だけでは足りない状況 → 生命倫理の4原則などの手引きが有効

II. 倫理理論（骨格）

看護倫理で特に重要な二理論：

  • 義務論（Deontology）：動機と規則
  • 帰結主義（Consequentialism）：結果と幸福の総量

義務論（カント）

  • 行為の善悪は内的動機で決まる
  • 状況に左右されない普遍的基準

人格（人間性）の尊重

  • 人を手段としてのみ扱ってはならない
  • 例：正当な報酬＝目的をもつ存在への敬意

自律（Autonomy）

  • 自己立法＝万人に適用可能な法を自らに課す
  • 普遍化可能性テスト（自殺の規則は普遍化できるか？）

帰結主義（功利主義）

  • 行為の善悪は結果で判定
  • 社会全体の幸福の総量を最大化
  • ベンサム：量（快楽↑ 苦痛↓） / ミル：質も考慮

医療での指標例

  • 平均寿命・5年相対生存率など
  • 幸福/コスト＝効率で比較（費用・人手・物資を分母）
表1-1相当の例示は口頭で。A:24.1% vs B:11.6% → 量ではA、費用考慮でBが有利の可能性。

応用と課題（バランス）

義務論の短所

  • 結果を軽視 → 向こう見ずな判断の危険

帰結主義の短所

  • 効率志向に偏る危険（人間の生を単純化）

生命倫理は両極の間でバランスを取る実践（後述の4原則もこの発想）。

事例：Aさんの告知問題

B看護師（義務論）

  • 仲間外れは間違い
  • 人格の尊重 / 自律の重視

C看護師（帰結主義）

  • 告知で抑うつ・希望喪失の恐れ
  • 結果の良否を予測して判断

実践では理念 × 結果予測の両立が望ましい。

III. ナラティブ倫理

なぜ必要か

  • 現実の課題は状況（文脈）に強く依存
  • 利害・価値観の不一致＝ナラティブの衝突
  • 対話による理解と調停が鍵

立場によって異なるナラティブ

患者

  • ライフヒストリーの断裂 → 新たな道筋探索

家族

  • 支援者/代理決定者になり得るが関係性は様々

医療従事者

  • 専門教育由来の疾病観・規範
  • 看護はアドボカシーを重視し、医師と患者の狭間で葛藤

事例：ホームレス男性の緊急手術

医師のナラティブ

  • 命を救うのが使命。反対されても見殺しにできない

妻のナラティブ

  • 15年の不在・苦労への憤りとむなしさ

相反するナラティブ → 倫理的ジレンマ。

対話と調停：実践ステップ

  1. 現状の客観把握（医学的状態・選択肢・社会状況）
  2. 関係者それぞれの思い・背景・価値観を傾聴
  3. ズレの可視化と、合意可能領域の探索
  4. 合意形成に向けた選択肢の再構成（説明責任）

補足：徳倫理

  • 「どう行動すべきか」よりどんな徳を備えるか
  • 例：忠義・仁愛・勇気・正直
  • 看護職：困難に向き合い、根拠を説明できる実践知
  • ※ 行為の具体指針は示さない → 当事者の熟慮が不可欠

IV. まとめ

  • 倫理学＝規範倫理学＋記述倫理学（両方重要）
  • 義務論：人格尊重・自律／動機
  • 帰結主義：結果・幸福総量／効率
  • 看護倫理＝両視点の統合とバランス
  • ナラティブ倫理：文脈理解・対話・調停で合意形成

→ 多様な倫理課題に対する実践知の第一歩。
