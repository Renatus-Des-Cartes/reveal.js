生命倫理 オンデマンド

脳死臓器移植(教科書pp.74--78)

担当：筒井 一穂

1. 臓器移植

定義

重い病気や事故などにより臓器の機能が低下した人に、他者の健康な臓器と取り替えて機能を回復させる医療。


原則

臓器の提供は、本人の意思 もしくは（本人の意思が不明である場合）家族の同意 に基づき行われる。

臓器移植のパターン

臓器を取り出すタイミングにより、移植後の生着率や対象臓器が異なる。

生体移植

生きている人の臓器を用いる

  • 腎臓
  • 肝臓 など

死体移植

亡くなった人の臓器を用いる

  • 心停止後：腎臓、眼球
  • 脳死後：心臓、肺、肝臓、膵臓、小腸

2. 倫理的問題の整理

  • ① 資源の配分は公正か？
    → 正義に基づく問い（限られた人しか移植を受けられない）
    → 生体移植・死体移植ともに当てはまる
  • ② 移植は人間を「資源」扱いするか？
    → 宗教・道徳観に基づく問い
    → 生体移植・死体移植ともに当てはまる
  • ③ 死体移植は「死」との関わり方として正しいか？
    → 移植待機者はドナーの死を望まざるを得ないのか？
    → 家族の穏やかな看取りを阻害する
  • ④ 脳死は「人の死」か？
    → 脳死状態は心臓移植のために作られた死ではないか？

3. 死の定義

「死とはなにか」に対する3つの説

一般的 全脳説

全身を統合的につかさどる脳機能が全面的に失われること（全脳機能の不可逆的停止）が個体の死。
→ いちばん納得しやすい定義。基本的に論争はない。

問題含み 高次脳説

思考・推論・感情などの高次脳機能が失われることを死とする。
→ 認知症や重度の精神疾患などが「死」とみなされかねない。

非脳説

脳以外の働き（呼吸、体液の流れ）、魂の離脱、関係の喪失などを死とする。
→ 生理学的根拠はないが、常識的・宗教的定義としては存在する。

4. 脳死の判定基準

「全脳機能の不可逆的停止」をどう確認するか？

① 旧来の判定法

  • 心拍動の停止
  • 呼吸の停止
  • 対光反射(瞳孔)の消失

② 現在の判定法

  • 深い昏睡
  • 瞳孔散大・固定
  • 脳幹反射の喪失
  • 平坦な脳波
  • 自発呼吸の停止

従来の判定法のポイント

  • 心拍動が停止していれば、必然的に脳へ酸素が行かず脳機能は停止する
  • 医学的にも確実であり、医学的知識やツールを持たない家族や関係者全員にとって、確認・納得しやすい判定法

現在の判定法のポイント

  • 人工呼吸器などの装置によって、心拍動・呼吸は継続する
専門的な手法を使うため、家族にとっては「死んでいる」かどうかが直感的にわからない。
生命維持治療の停止が尊厳"死"なら、生命維持治療をしている間は"死"ではないのでは？

なぜ現代の判定法を用いるのか？

従来の判定法では心臓移植が困難になる

論理的な包含関係

従来の判定法の「死」は現在の判定法よりも狭い。つまり「従来の基準では死ではないが、現在の基準では死である」ケースがある。

心臓移植の必要性

心臓が完全に停止すると、移植後の生着率が著しく下がる。現在の判定法なら新鮮な心臓の移植が可能。

移植のための死？

批判：「現在の判定法は、移植を可能にするために作られた死ではないか？」

反論：これはあくまで医学的な「判定法」であり、「死」の定義そのものとは関係がない。
（メディア報道などでは、判定法と定義が混同されることが多い）

同意の方式

オプトイン方式：旧法（1997）の同意システム。本人のドナーカードおよびリヴィング・ウィルに基づき摘出を行う

オプトアウト方式：新法（2009）の同意システム。本人が反対していないかぎり、家族の同意があれば摘出を行うことができる
